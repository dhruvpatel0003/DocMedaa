# DocMedaa – Smart Patient Portal & Healthcare Companion

## Overview
DocMedaa is a unified healthcare platform designed to improve communication, engagement, and health tracking between patients and healthcare providers. The application combines appointment management, secure messaging, educational content, and smartwatch-based health monitoring into a single, intuitive system.

The project was developed as a full-stack software solution with a focus on usability, scalability, and real-world healthcare challenges.

## Key Features
- Secure patient–doctor messaging
- Appointment scheduling with automated reminders
- Educational articles and videos managed via admin portal
- Smartwatch integration using Google Fit
- Heart-rate and activity tracking
- Real-time push notifications
- Role-based access for patients, doctors, and admins

## Tech Stack
**Frontend (Mobile):**
- Flutter (Dart)
- BLoC state management

**Frontend (Admin Panel):**
- React
- Redux

**Backend:**
- Node.js
- MongoDB

**APIs & Services:**
- Google Fit API
- Firebase Cloud Messaging
- JWT Authentication
- Google Cloud Deployment
